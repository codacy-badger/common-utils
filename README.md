## Common Utils

A small java project holding various utility classes to reduce development effort. 

### Target Version

`0.0.1`

### Setup

Add dependency in your `pom.xml`

```markdown

<dependency>
	<groupId>com.common.util</groupId>
	<artifactId>common-utils</artifactId>
	<version>${version}</version>	
</dependency>

```

### Build

```markdown
mvn clean install
```
